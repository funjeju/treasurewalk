# 02 · 게이미피케이션 시스템

이 앱의 모든 게임 요소는 **Octalysis 8 코어 드라이브**(Yu-kai Chou)에 매핑된다. 목표: 단순 PBL(포인트·뱃지·리더보드)을 넘어 **걷기라는 현실 행동을 자발적으로 반복**하게 만드는 것. (출처: yukaichou.com / Octalysis Group / Wikipedia.)

> 설계 철학: **White Hat(자율·성취·의미)을 척추로, Black Hat(희소·호기심)을 소량의 양념으로.** 블랙햇 과용은 "끝나면 떠나고 싶은" 강박을 만든다(Octalysis 경고).

| # | 코어 드라이브 | 우리 앱에서의 구현 | 티어 적용 |

| **CD3 창의·피드백**(Empowerment) | 아바타·베이스캠프 꾸미기, 보물 숨길 위치를 부모가 창작, 발견 경로 자유 선택 | T2+ |
| **CD4 소유·획득**(Ownership) | 코인/젬, **보물 도감(컬렉션)**, 세트 완성, 수집형 카드, 한정 스킨 | 전 티어 |
| **CD5 사회·관계**(Social) | 가족 피드("하준이 보물 발견!"), 형제 협동/경쟁 랭킹, 인증서 공유, 부모 칭찬 리액션 | T2+ (가족 한정) |
| **CD6 희소·조급**(Scarcity) | 시즌 한정 보물, 주말 특별 미션, 스트릭 유지, 일일 리셋 퀘스트 | 소량, T3+ |
| **CD7 예측불가·호기심**(Curiosity) | "미스터리 상자"(보상 랜덤), 깜짝 보너스, 안개 너머 미발견 보물 실루엣, 이스터에그 | 전 티어 |
| **CD8 손실·회피**(Loss/Avoidance) | 스트릭 끊김 경고(부드럽게), 시즌 종료 카운트다운 | **아주 절제**, T3+ 만 |

> **균형 규칙:** CD2·CD4(척추) 항상 ON → CD1·CD3(의미·창의)로 깊이 → CD5·CD7(사회·호기심)로 재미 → CD6·CD8(희소·손실)은 한 스푼. 아동 티어(T1/T2)는 CD8을 거의 쓰지 않는다(불안 유발 회피, `docs/07` 정신건강 고려).

## 3. "진짜 게임처럼" 몰입 장치 (Game Feel / Juice)

게임이 재밌게 *느껴지는* 건 콘텐츠가 아니라 **피드백 밀도**다. 모든 핵심 동작에 3겹(시각·청각·촉각)을 얹는다.

### 3.1 지도를 게임 월드로 (최대 레버)
- 양피지/판타지 벡터 스킨(MapLibre), **포그 오브 워**(미탐험은 안개 → 걸으면 열림), 캐릭터 아바타가 실제로 걸어다님, 보물=빛나는 X/상자.

### 3.2 걷기를 게임으로 — 근접 메커닉 (GPS 한계를 재미로 역전)
- **뜨거워/차가워 미터**(거리→색·진동 강도), **줄어드는 거리 링**, **보물 레이더 핑**(가까울수록 간격↓), **나침반 화살표**(방향만).
- 구현: 기기 위치와 보물 좌표의 Haversine 거리 → 0~1 정규화 → 미터/햅틱/사운드에 바인딩. (`docs/03 §geo`)

### 3.3 발견의 순간 = 히어로 연출 (예산 절반 투입)
- 화면 전환 → **상자 등장 → 탭해서 열기(Rive) → 빛+코인 분수(Lottie) → "+500원" 튀어오름 → 팡파르 + 강한 햅틱.**

### 3.4 성장·수집 시각화 (업로드 레퍼런스 차용)
- **보물 도감**(빈 칸이 수집욕 자극), **세트 완성**("Set Completed!" 연출), **카드형 컬렉션**, **시즌 패스 트랙**, **월드맵 경로(레벨 노드)**.

### 3.5 메타 화면 (레퍼런스 → 보물찾기 번역)
| 레퍼런스 요소 | 우리 화면 |
|---|---|
| Announcement 모달 | 시즌/이벤트 공지 |
| Easter Pass(배틀패스) | **탐험 패스**(무료 트랙 + 시즌 보상) |
| Shop / Best Deals / Daily Bling | **보물상점**(코스메틱 위주) / **일일 퀘스트** |
| Collections / Card Stars | **보물 도감** / **별 보상** |
| Set Completed! | **세트 완성** 연출 |
| Continue?(리바이브 타이머) | **힌트 더 받기**(보물 못 찾을 때) |
| 하단 진행 경로 | **월드맵 보물 경로** |

> ⚠️ 상점/패스에서 **실제 결제**는 P1 범위 아님. 코인은 "탐험으로 버는 게임 내 화폐"로 시작. 현금 결제·실물 보상 자동화는 미결(`docs/07`).

## 4. 보상 경제 (Economy) — P1은 단순, 확장 자리만

- **용돈(현실 보상):** 부모가 보물마다 지정 → 발견 시 "요청" → 부모 수동 지급. (자동 이체 ❌)
- **코인/젬(게임 내):** 발견·일일퀘·스트릭으로 적립 → 아바타/스킨 해금. (P1: 코인만, 젬은 자리만)
- **XP/레벨:** 거리·발견·연속성 누적.
- 확장 enum: `FIXED → DISTANCE_SCALED → STREAK_BONUS`(`docs/04`).

## 5. 안티패턴 (하지 말 것) — 중요

1. **과잉정당화 효과 주의:** 재미있던 활동에 돈을 과하게 걸면 보상 없을 때 안 하게 된다. → **재미·수집을 주연, 돈을 조연**으로. 코인/도감/연출을 전면에, 용돈은 보너스처럼.
2. **블랙햇 과용 금지:** 상시 손실 공포, 강제 스트릭 압박, 페이투윈, 다크패턴 → 아동 대상에서 특히 금지.
3. **현금 ↔ 개인정보 교환형 설계 금지:** 한국 가이드라인이 "현금/아이템 대가로 개인정보 입력시키는 설계 자제" 권고(`docs/07`). 용돈은 **부모가 자녀에게 주는 것**이지, 서비스가 데이터 대가로 지급하는 구조가 아니어야 한다.
4. **정신건강 고려:** 부정적 자기비교를 부추기는 랭킹/스트릭 표현 회피. 실패를 "다음 기회"로 프레이밍.
